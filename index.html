FastAPI · JWT

Lockbox

Production-ready Authentication Microservice built with FastAPI (fully async) featuring JWT access tokens, rotating refresh tokens with DB-backed revocation, and bcrypt password hashing — covering the complete auth lifecycle securely. Hardened against brute-force attacks via IP-based rate limiting, with tokens automatically invalidated on logout, ensuring zero token reuse and stateless-yet-secure session management. Ships with auto-generated Swagger UI, a full pytest test suite, and runs anywhere from a cloud server to a phone via Termux — plug-and-play auth backend ready for any application stack.

ML · CNN · CIFAR-10

Imarec

ImaRec is a full-stack Image Recognition Web App powered by a custom-trained Convolutional Neural Network (CNN) on the CIFAR-10 dataset, capable of classifying uploaded images into 10 categories — from animals to vehicles — directly in the browser. Built with Flask + TensorFlow/Keras, it accepts image uploads, runs them through the trained CNN model, and returns a real-time classification result via a clean HTML/CSS frontend — no external API, fully self-contained. Deployable anywhere with Gunicorn + Procfile support, and built with OpenCV for image preprocessing — a complete end-to-end ML pipeline from model training to web serving in one repo.

AI/ML · Keras · Tensorflow · NLP

DocBot

DocBot is an AI-powered Medical Chatbot that understands natural language to assist users with symptom checking, medical queries, and health consultations — powered by a custom-trained Neural Network (Keras .h5 model) with intent-based NLP classification. Built with Python + TensorFlow/Keras, it preprocesses user input using tokenization and lemmatization (stored via .pkl files), maps it to predefined medical intents from a structured intents.json, and delivers contextual responses — all without any external API dependency. Served through a clean, interactive Streamlit web UI, the entire ML pipeline — from training to inference — lives in a single repo, making it instantly runnable with one command: streamlit run chatbot.py.
